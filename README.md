# 🗑 Waste Classification System

This project is a **Deep Learning-based Waste Classification System** that classifies different types of household waste using images.

It uses **Convolutional Neural Networks (CNN)** and **Transfer Learning (MobileNetV2)** to achieve accurate predictions. The system is deployed using a **Streamlit web application** for real-time image classification.

---

## 🚀 Features
- Image classification using Deep Learning
- Custom CNN model (baseline)
- Pretrained MobileNetV2 model (high accuracy)
- Streamlit web-based GUI
- Real-time image upload and prediction
- Waste category mapping (Recycle / Reuse / Reduce)

---

## 🧠 Models Used
- Convolutional Neural Network (CNN)
- MobileNetV2 (Transfer Learning)

---

## 📊 Dataset

The dataset used is **Recyclable and Household Waste Classification**, containing multiple waste categories.

Each class contains:
- 250 images (default)
- 250 images (real_world)  
👉 Total: **500 images per class**
